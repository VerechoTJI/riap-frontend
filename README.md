# Frontend Repo

This repository is reserved for the RIAP frontend application.

## Purpose

- Keep UI-specific code isolated from backend services.
- Build and test the frontend in its own repo.

## Status

- Placeholder repo scaffolded for future frontend work.

## Quick setup (when frontend code is present)

Typical steps once frontend sources are added:

```powershell
# install Node.js (LTS) and a package manager (npm comes with Node)
# from repo root
cd frontend
# install dependencies
npm install
# start dev server
npm run dev
# build production bundle
npm run build
```

If you prefer pnpm:

```powershell
pnpm install
pnpm run dev
```
